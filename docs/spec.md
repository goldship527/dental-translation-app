# dental-translation-app 仕様書

## 目的
- 歯科医院で外国語話者に短い案内を行うための翻訳補助Webアプリを作る。
- まずは受付、問診、治療説明、会計、術後説明で使う定型文を素早く表示できる状態を目指す。

## 想定ユーザー
- 歯科医院の受付スタッフ
- 歯科衛生士
- 歯科医師
- 日本語が十分に伝わりにくい患者への説明を補助したい院内スタッフ

## 必須機能
- [x] 日本語の短文を入力できる
- [x] 翻訳先言語を選べる
- [x] 歯科医院での場面を選べる
- [x] デモ翻訳結果を表示できる
- [x] よく使う歯科フレーズを選択できる
- [x] 個人情報を入力しない注意を画面に表示する

## できれば欲しい機能
- [ ] 実翻訳APIとの連携
- [ ] よく使うフレーズのカテゴリ追加
- [ ] フレーズのお気に入り登録
- [ ] 音声読み上げ
- [ ] 多言語の表示品質チェック
- [ ] スマホ・タブレットでの実機確認

## 画面
- メイン画面
  - 場面選択
  - 翻訳先言語選択
  - 日本語入力欄
  - 翻訳結果表示
  - よく使うフレーズ一覧
  - 個人情報入力禁止の注意表示

## データ項目
- 入力文
- 翻訳先言語
- 歯科場面
- デモ翻訳文
- 定型フレーズ

## セキュリティ注意点
- APIキー、パスワード、個人情報を保存しない
- 患者情報は扱わない
- 会社機密情報を含めない
- 実翻訳APIを使う場合、APIキーは環境変数またはGit管理外のローカル設定に置く
- 入力内容を外部APIへ送信する場合は、患者情報を入力しない運用を画面とREADMEに明記する

## AIへの依頼
この仕様をもとに、まずはローカルで表示できる静的Webプロトタイプから実装する。
実翻訳API連携は、画面と操作感を確認した後の次フェーズで検討する。
